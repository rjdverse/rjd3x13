# Java Utility Functions

These functions are used in all JDemetra+ 3.0 packages to easily
interact between R and Java objects.

## Usage

``` r
.x13_rslts(jrslts)

.jd2r_spec_x11(jspec)

.r2jd_spec_x11(spec)

.r2jd_spec_regarima(spec)

.jd2r_spec_regarima(jspec)

.r2jd_spec_x13(spec)

.jd2r_spec_x13(jspec)

jversion
```

## Format

An object of class `integer` of length 1.

## Arguments

- spec, jspec, jrslts:

  parameters.
