# X-11 Decomposition Algorithm

X-11 Decomposition Algorithm

## Usage

``` r
x11(ts, spec = x11_spec(), userdefined = NULL)
```

## Arguments

- ts:

  an univariate time series.

- spec:

  the specification.

- userdefined:

  a vector containing additional output variables (see
  [`x13_dictionary()`](https://rjdverse.github.io/rjd3x13/reference/x13_dictionary.md)).

## Value

the `x11()` function returns a list with the results (series) and final
parameters
