# rjd3x13: Seasonal Adjustment with 'X-13' in 'JDemetra+' 3.x

R Interface to 'JDemetra+' 3.x (<https://github.com/jdemetra>) time
series analysis software. It offers full access to options and outputs
of 'X-13', including Reg-ARIMA modelling (automatic AutoRegressive
Integrated Moving Average (ARIMA) model with outlier detection and
trading days adjustment) and X-11 decomposition.
